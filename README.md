# Physical-Design-Synthesis-of-Synchronous-FIFO
Designed and verified a 500 MHz Synchronous FIFO using Verilog, Xcelium, and SimVision for pointer/flag debugging. Synthesized RTL in Cadence Genus with custom SDC constraints and resolved scan-cell mapping issues. Completed full physical design flow in Innovus including floorplanning, power planning, routing, and timing closure with OCV/CPPR.
